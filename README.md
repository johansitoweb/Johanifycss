## como centralizar un texto con Johanifycss.
```html
<!-- index.html -->
<html>
  <head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/johansitoweb/vs1.0@main/framework-Johanifycss-version1.0.css">
  </head>
  <body>
            <h1 class="text-center">texto</h1>
</body>
</html>
```
